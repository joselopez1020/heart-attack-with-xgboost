# Enfermedad-xgboost

Predicción de Riesgo de Ataque al Corazón

En este portafolio crearemos un modelo que nos ayudara a predecir y detectar un problema importante de salud, haremos una predicción para saber su el paciente sufrirá un ataque al corazón utilizando un conjunto de variables clínicas como age,	sex,	resting_bp,	cholesterol,	fasting_bs,	ecg_result,	max_heart_rate,	exercise_angina,	st_depression,	slope,	num_major_vessels,	thalassemia,	heart_attack

Para crear el modelo utilizamos:

Python · Pandas · NumPy · scikit-learn · Matplotlib · Seaborn · os · random · shape · xgboost 

El dataset utilizado en el modelo se encuentra en Kaggle

Link: https://www.kaggle.com/datasets/payaldhokane/heart-attack-risk-analysis
